## ğŸš€ How to Use
Visual recognition is performance-intensive. A PC capable of running Genshin Impact smoothly at **Medium settings, 60 FPS** is recommended for the best experience.

**System Requirements:**
*   Windows 10/11 (64-bit)
*   [.NET 8 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/latest/runtime) (Installer will prompt if missing)

**âš ï¸ Important Notes:**
1.  Restart the app after changing **window size, resolution, or monitor**.
2.  **Do not use any screen filters** (HDR, NVIDIA Freestyle, etc.). Keep in-game brightness default.
3.  Currently supports **16:9** aspect ratios only. **1920x1080 windowed mode is recommended.**
4.  **Input simulation may be blocked by security software.** Add to whitelist. Known issues with 360 Security or custom Windows Defender rules.

**After launching, select your capture method on the "Start" page and click "Start" to enjoy BetterGI!**

Detailed guide: [Quick Start](https://bettergi.com/quickstart.html)
